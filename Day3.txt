2. semi-structured data  eg: powerpoint presentation

***5 V's of big data***

1. Velocity- it refers to speed at which data travels from x node to y server in a particular network.
2. Volume- it is refered as 1Mb/s
3. Value- data which is sent should have some value for organisation.
4. Verasity- when data trvels in network data loss takes place.
5. Variety- data should be of varieties and gernes.
